CivilSurvey

CivilSurvey is an open source, lightweight survey engine built with Angular.js, driven by a JSON API.

Features

  • All major question types: text line, textarea, select dropdown, radio buttons, checkboxes
  • Built-in validation for numerous situations
  • Detail box ("Other/Please Specify")
  • Randomize order of options (allows exemptions, for example "All/None of the Above")
